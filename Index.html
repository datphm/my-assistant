iPhone / iPad

Hồ sơ My Assistant

Hồ sơ tạo một icon My Assistant trên Màn hình chính và mở thẳng đúng bản Google Apps Script này.

Tải hồ sơ iPhone

Phải mở link bằng Safari. Sau khi tải: Cài đặt iPhone → Đã tải về hồ sơ → My Assistant → Cài đặt. File không còn trường Icon rỗng từng gây lỗi trước đây.

Android

Thêm vào màn hình chính

Trên Android: mở trang này bằng Chrome → menu ⋮ → Thêm vào màn hình chính.

Đây là web app; chưa có APK native ký riêng.
