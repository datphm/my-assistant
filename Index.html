Đồng bộ email ngân hàng

Quét Gmail theo truy vấn bạn tự chọn. Nếu email có “Số dư”, app cập nhật số dư; nếu là “ghi có/received/refund” thì cộng tiền, còn giao dịch chi thì trừ tiền.

Android

APK / cài nhanh

Bản web dùng được ngay bằng Chrome → Add to Home screen. APK native cần build riêng từ repo.

Chỗ tải APK / hướng dẫn

Mình đã để sẵn “chỗ tải”; khi build APK thật, nút này sẽ trỏ sang file APK.
